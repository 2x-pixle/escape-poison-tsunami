# Escape Poison Tsunami

## Game Description
Escape Poison Tsunami is an exhilarating adventure game where players must navigate through treacherous terrains while avoiding deadly poisonous tsunamis. The primary objective is to survive as long as possible and reach safe zones.

## Features
- **Dynamic Environments**: Experience changing landscapes and obstacles.
- **Challenging Levels**: Progress through multiple levels with increasing difficulty.
- **Power-ups**: Collect power-ups that grant temporary abilities to enhance survivability.
- **Leaderboard**: Compete with friends and see who can survive the longest.

## Controls
- **Arrow Keys**: Move the character (Up, Down, Left, Right).
- **Space Bar**: Jump over obstacles.
- **P**: Pause the game.
- **R**: Restart the level.

## How to Play
1. Start the game by pressing the 'Start' button on the main menu.
2. Use the arrow keys to navigate your character.
3. Avoid getting caught in the poisonous tsunami waves.
4. Collect power-ups to gain advantages during your escape.
5. Aim to reach the safe zone before the time runs out!

Good luck, and may you escape the poison tsunami!
